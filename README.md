<h1 align="center">🚀 Reridex</h1>

<h3 align="center">
Smart Vehicle Service & Lead Management Platform
</h3>

<p align="center">
A production-ready full stack vehicle management platform built using Spring Boot, React, MySQL, and cloud integrations for scalable lead tracking and inventory management.
</p>

---

## ✨ Key Features

✅ Bike Listing & Inventory Management  
✅ Multi-Channel Lead Capture System  
✅ Vehicle Valuation Requests  
✅ Test Drive Booking Workflow  
✅ Role-Based Admin Dashboard  
✅ Secure Authentication & Authorization  
✅ Cloudinary Image Management  
✅ Google Sheets Lead Automation  
✅ Automated Email Notifications  
✅ Location-Based Vehicle Filtering  
✅ RESTful API Architecture  
✅ Dockerized Deployment Setup

## 🧠 System Workflow

```text
User Visits Platform
        ↓
Browse Bikes / Submit Leads
        ↓
React Frontend Sends API Requests
        ↓
Spring Boot Backend Processes Data
        ↓
MySQL Stores Records
        ↓
Cloudinary Handles Media Uploads
        ↓
Google Sheets Updates Lead Tracker
        ↓
Email Notifications Triggered
```

## 🎯 Problems It Solves

ReriDex streamlines vehicle lead management and inventory handling by replacing fragmented workflows with a centralized, scalable platform.

The platform:
- automates lead collection
- simplifies inventory scaling
- centralizes operational tracking
- improves media handling efficiency
- reduces manual admin overhead
